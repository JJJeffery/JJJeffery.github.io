类型规范

Python最开始让人舒服的是不用事先定义变量，也不用声明变量种类。

但有时候这种自由会导致一些乱七八糟的结果，这时候就有点想念Java的类型声明了。

好在Python其实也有变量声明，不过不是强制的，即使错了也能运行，不会报错。

但IDE会高亮提示，在大型项目中应该还是很有用的。
